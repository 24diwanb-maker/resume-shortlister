# Smart Resume Shortlisting System
##  Overview
A Streamlit-based web application that automates the hiring process using Design and Analysis of Algorithms (DAA). It selects optimal candidates based on skills, experience, and salary constraints.
---
##  Features
- Add and manage candidate details  
- Sort candidates by performance  
- Budget-based hiring optimization  
- Compare Greedy vs Dynamic Programming  
- Graphical visualization of results  
---
##  Algorithms Used
- **Merge Sort (Divide & Conquer)** – for ranking candidates  
- **Greedy Algorithm** – fast selection based on score/salary  
- **Knapsack (Dynamic Programming)** – optimal selection under budget  
---
##  Tech Stack
- Python  
- Streamlit  
- Pandas  
- Matplotlib  
---
##  How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
